# RBS Hacker Rank Task

## Task 1
Given a set of numbers, find the total pairs of numbers which has a difference of K.

### Usage

```
[SIZE OF SET] [K]
[SET OF NUMBERS WITH A SPACE BETWEEN EACH NUMBER]
```

### For example

[Input]
```
5 2

1 4 3 5 2
```

should give an output of:
```
3
```

where K = 2
